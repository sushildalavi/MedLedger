Chain integrity

Each node replays its local hash chain, verifies its Ed25519 signatures, and the gateway compares block_hash across nodes. A single tampered node is detected by majority disagreement.

What's actually written to disk. patient_id, user_id, action, and event timestamps are inside the AES-256-GCM ciphertext — never in plaintext on disk.
